# FashionMNIST 卷积自编码器实现
本项目基于PyTorch实现了**卷积自编码器(Convolutional Autoencoder, CAE)**，并在FashionMNIST数据集上完成训练与验证，实现了图像重建、隐空间可视化、模型结构可视化等功能，可作为自编码器入门学习的参考实现。

## 项目简介
自编码器是一种无监督学习的神经网络，由**编码器(Encoder)**和**解码器(Decoder)**两部分组成：
- 编码器：将高维图像数据压缩为低维隐空间特征
- 解码器：将低维隐空间特征重建为原始维度的图像
本项目采用**卷积+转置卷积**结构替代全连接层，更适合处理图像数据，在FashionMNIST（28×28灰度服饰图像）上实现了高效的特征压缩与图像重建，隐空间维度设为2维，方便可视化分析。

### 项目亮点
1. 纯卷积结构，贴合图像数据的空间特征提取
2. 完整的训练/验证/测试流程，包含损失可视化
3. 支持图像重建效果对比、2维隐空间可视化
4. 提供ONNX模型导出与Netron交互式结构可视化
5. 兼容CPU/GPU自动适配，训练效率高

## 数据集介绍
使用**FashionMNIST**数据集，包含10类服饰的灰度图像，共70000张（60000张训练集，10000张测试集），图像尺寸为28×28×1。
- 数据集会通过`torchvision`自动下载，保存至`./data`目录
- 训练集按8:2划分为训练集和验证集，用于模型训练与早停验证
- 数据变换仅做`ToTensor()`，将像素值归一化至[0,1]区间

## 模型结构
### 编码器(Encoder)
采用**3层卷积+2层全连接**结构，逐步压缩图像维度至2维隐空间：
1. 卷积层：提取图像空间特征，逐步降低分辨率、增加通道数
2. 展平层：将卷积特征图转为一维向量
3. 全连接层：将高维特征向量压缩为2维隐空间特征

| 层类型       | 层参数                | 输出尺寸       |
|--------------|-----------------------|----------------|
| Conv2d       | 1→8, 3×3, stride=2, pad=1 | (8,14,14)      |
| Conv2d       | 8→16, 3×3, stride=2, pad=1 | (16,7,7)       |
| Conv2d       | 16→32, 3×3, stride=2, pad=0 | (32,3,3)       |
| Flatten      | start_dim=1           | (288,)         |
| Linear       | 288→128               | (128,)         |
| Linear       | 128→2                 | (2,)（隐空间） |

### 解码器(Decoder)
采用**2层全连接+3层转置卷积**结构，将2维隐空间特征重建为28×28图像：
1. 全连接层：将低维隐空间特征还原为高维向量
2. 反展平层：将一维向量转为卷积特征图尺寸
3. 转置卷积层：逐步上采样特征图，降低通道数，恢复原始图像尺寸
4. Sigmoid激活：将输出像素值限制在[0,1]，与输入数据匹配

| 层类型           | 层参数                | 输出尺寸       |
|------------------|-----------------------|----------------|
| Linear           | 2→128                 | (128,)         |
| Linear           | 128→288               | (288,)         |
| Unflatten        | (32,3,3)              | (32,3,3)       |
| ConvTranspose2d  | 32→16, 3×3, stride=2  | (16,7,7)       |
| ConvTranspose2d  | 16→8, 3×3, stride=2, pad=1, output_pad=1 | (8,14,14) |
| ConvTranspose2d  | 8→1, 3×3, stride=2, pad=1, output_pad=1 | (1,28,28) |
| Sigmoid          | -                     | (1,28,28)      |

## 训练设置
### 超参数
| 超参数           | 取值       | 说明                     |
|------------------|------------|--------------------------|
| 隐空间维度       | 2          | 方便可视化               |
| 学习率           | 0.001      | Adam优化器学习率         |
| 批次大小(batch_size) | 256    | 平衡训练效率与显存       |
| 训练轮数(num_epoch) | 1       | 快速验证，可自行增加     |
| 权重衰减         | 1e-5       | 防止过拟合               |
| 损失函数         | MSELoss    | 图像重建的像素级损失     |
| 优化器           | Adam       | 自适应学习率，训练稳定   |

### 训练流程
1. 设备自动适配：优先使用GPU(cuda)，无GPU则使用CPU
2. 模型初始化：编码器和解码器实例化并移至指定设备
3. 数据加载：训练集/验证集/测试集通过DataLoader批量加载
4. 训练循环：
   - 训练模式：计算重建损失，反向传播更新参数
   - 验证模式：关闭梯度计算，评估模型在验证集上的泛化能力
5. 损失记录：保存每轮训练/验证损失，用于可视化

## 核心功能模块
### 1. 数据加载与划分
```python
# 自动下载数据集
train_dataset = datasets.FashionMNIST(root='./data', train=True, transform=transforms.ToTensor(), download=True)
# 训练集按8:2划分为训练/验证集
train_data, val_data = random_split(train_dataset, [int(m-m*0.2), int(m*0.2)])
# 数据加载器
train_loader = DataLoader(dataset=train_data, batch_size=256, shuffle=True)
```

### 2. 模型训练与验证函数
- `train_epoch()`：单轮训练，计算批次损失，反向传播更新参数
- `test_epoch()`：单轮验证/测试，关闭梯度计算，评估全局重建损失

### 3. 结果可视化
#### 3.1 训练损失可视化
绘制训练集与验证集的损失曲线，观察模型收敛情况
#### 3.2 图像重建效果对比
随机选取测试集图像，对比**原始图像**与**模型重建图像**，直观评估重建效果
#### 3.3 隐空间网格重建
在2维隐空间的指定区间内生成网格点，将每个网格点通过解码器重建为图像，展示隐空间的特征分布与连续性
#### 3.4 隐空间分布分析
将测试集所有图像编码至2维隐空间，绘制散点图并按类别着色，观察不同类别在隐空间的聚类效果

### 4. 模型结构可视化
项目提供两种模型结构查看方式：
1. `torchsummary`：打印模型层结构、输出尺寸、参数数量（控制台输出）
2. `Netron`：导出ONNX格式模型，通过浏览器交互式查看模型结构（需取消代码注释）

```python
# 取消代码末尾注释，执行后自动打开浏览器可视化模型
x = torch.randn(1,1, 28, 28).to(device)
model = nn.Sequential(OrderedDict([('encoder', encoder), ('decoder',decoder)]))
torch.onnx.export(model, x, "model.onnx", input_names=['input'], output_names=['output'])
netron.start("model.onnx", browse=True)
```

## 扩展与优化
1. **增加训练轮数**：将`num_epoch`调至20~50，模型重建效果会进一步提升
2. **加入早停机制**：当验证集损失连续多轮不下降时停止训练，防止过拟合
3. **调优超参数**：调整学习率、批次大小、隐空间维度，探索更优的模型性能
4. **实现变分自编码器(VAE)**：在现有基础上修改编码器，输出均值和方差，引入KL散度损失，实现隐空间的正态分布约束，提升生成能力
5. **数据增强**：对训练集加入随机旋转、平移等数据增强，提升模型泛化能力
6. **模型轻量化**：调整卷积层通道数，减少模型参数，提升训练和推理速度
